## Currently Building

### [iLity Hub](https://github.com/kaywalker91) — Multi-chain Crypto Wallet & Social Trading

Building a trustless social platform where every post is backed by on-chain transactions.

- **50,000+ LOC** across 12 modules with Clean Architecture
- **8 wallets supported** via WalletConnect v2 (MetaMask, Trust Wallet, etc.)
- **3 EVM networks**: Ethereum, BNB Chain, Base
- **299 test cases** with state machine-based wallet connection flow

Tech: `Flutter` `Riverpod 3.x` `WalletConnect v2` `reown_appkit` `fl_chart`

## Featured Work

### Emergency Ready App (안전디딤돌)
**Korea's official disaster safety app** — Ministry of the Interior and Safety

Migrated legacy Android (Java) + iOS (Objective-C) to Flutter.

- 19 languages with real-time translation (Google Translate API V3)
- Location-based FCM auto-subscribe with GPS drift correction
- Priority queue for emergency message delivery
- Full TalkBack/VoiceOver accessibility

### MindLog
**AI-powered emotional diary** — [Google Play](https://play.google.com/store/apps/details?id=com.kaywalker.mindlog)

Llama 3.3 analyzes journal entries and provides personalized support.

- 2-layer safety filter (input keywords + AI response validation)
- Crisis detection with immediate hotline connection
- 6 production releases

### TimeWalker
**Interactive history education game** — [Live Demo](https://kaywalker91.github.io/TimeWalker/)

Time travel meets map exploration. Built with Flame engine.

- Branching dialogue system with historical figures
- Offline-first with Supabase cloud sync
